Computer Science Principles

September 20th - October 5th 2018

The objective of this project was to create and improve an app through App Inventor. After finishing the tutorial, my partner and I added many other features such as being able to rotate pictures, being able to draw on them, and being able to delete and save them. While my partner and I were working on our project, we faced many problems. Having enough space for all of the buttons and not being able to control which box the image went in were only two of problems that we had. Through this project, I learned how to code and create an app through a program that I have never used before.
